PHPUnit Hands-on
================================================================================


Requirements
--------------------------------------------------------------------------------

- Docker for Mac


Environment
--------------------------------------------------------------------------------

- PHP 7
- PHPUnit 6.0.7


Getting Started
--------------------------------------------------------------------------------

Clone this repository:

```
$ git clone https://github.com/Tomohiro/phpunit-handson
```

Initialize:

```sh
$ ./script/setup  # Setup dependency docker container and composer packages
```


Testing
--------------------------------------------------------------------------------

Run phpunit:

```sh
$ ./bin/phpunit -c phpunit.xml
```
