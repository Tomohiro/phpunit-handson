PHPUnit Handson
================================================================================


Requirements
--------------------------------------------------------------------------------

- Docker for Mac


Environment
--------------------------------------------------------------------------------

- PHP 7
- PHPUnit 6.0.7


Getting Started
--------------------------------------------------------------------------------

Clone this repository:

```
$ git clone https://github.com/Tomohiro/phpunit-handson
```

Bootstrap:

```sh
$ ./script/bootstrap
```


Testing
--------------------------------------------------------------------------------

Run phpunit:

```sh
$ ./bin/phpunit -c phpunit.xml
```
